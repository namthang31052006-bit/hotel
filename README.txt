LUMISTAY - WEBSITE ĐẶT PHÒNG KHÁCH SẠN

Cách chạy:
1. Giải nén file hotel-booking.zip.
2. Mở file index.html bằng Chrome, Edge hoặc Firefox.

Cấu trúc:
- index.html: Nội dung website.
- style.css: Giao diện và responsive.
- script.js: Tìm kiếm, lọc, lưu phòng và đặt phòng.
- assets/: Ảnh khách sạn.

Responsive theo yêu cầu:
- Mobile: 1 thẻ phòng mỗi hàng.
- Tablet từ 772px: thẻ rộng 45vw, ảnh cao tối đa 30vh, 2 thẻ mỗi hàng.
- Màn hình lớn từ 998px: thẻ rộng 30vw, 3 thẻ mỗi hàng.
